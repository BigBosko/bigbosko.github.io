If, else in else if stavki so logični stavki, ki jih uporabljamo za reševanje problemov in logični potek kode.

1. If stavek

If stavek se uporabljaja za izvajanje neke kode, če je nek zastavljen pogoj pravilen. Da začnemo if stavek, moramo zapisati ključno besedo »if« in nato v oklepaju določiti pogoj. Nato znotraj zavitih oklepajev zapišemo kaj se zgodi če je pogoj pravilen.

2. Else stavek

Else stavek se uporablja po if stavku. Else stavek izvede neko kodo če je bil pogoj if stavka napačen. Else stavek zapišemo z ključno besedo »else«. Tukaj ne določimo pogoja, saj se izvede če je bil pogoj if stavka napačen.

3. Else if stavek

If stavek se uporablja kot drugi if stavek po prvem if stavku. Če prvi pogoj ni pravilen, lahko v else if stavku zastavimo še en pogoj pod katerim se bo izvedla neka druga koda. Else if stavek zapišemo z ključno besedo »else if«

Določil sem spremenljivko »stevilo« in ji določil vrednost 20. Nato sem napisal if stavek. Pogoj da se izvede je, da je vrednost spremenljivke »stevilo« 20. Če je pogoj if stavka pravilen, se izpiše: »Stevilo je manjse od 20«. Nato sem zapisal else if stavek. Pogoj da se izvede je, da je spremenljivka »stevilo« manjša od 20. Če je pogoj pravilen, se izpiše: »Stevilo je manjse od 20. Na koncu sem zapisal še else stavek. Ko se izvede else stavek, se izpiše: »Število je večje od 20«. Ker je bila vrednost spremenljivke manjša od 20, se je izvedel else if stavek.
